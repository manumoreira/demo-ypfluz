## Como verlo localmente

1. Clonar el código:

   ```sh
   $ git clone https://github.com/streamlit/demo-stockpeers.git
   ```

2. Iniciar el entorno virtual y las dependencias (requiere uv):

   ```sh
   $ uv venv

   $ .venv/bin/activate

   $ uv sync
   ```

3. Iniciar la app:

    ```sh
    $ streamlit run streamlit_app.py
    ```
